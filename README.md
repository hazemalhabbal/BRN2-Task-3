# Amazon Air Optimization

This project solves a transportation optimization problem using linear programming in Python.

## Objective
Minimize total shipment cost while satisfying demand and capacity constraints.

## Files
- amazon_air_optimization_clean.py: Python optimization model
- amazon_air_optimization_real_output.txt: Solver output showing optimal solution

## How to Run
python3 amazon_air_optimization_clean.py

## Result
The solver returns an optimal solution with total cost of 91,717.35.
